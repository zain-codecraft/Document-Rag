# KRWORK Frontend

React frontend for the KRWORK RAG assistant.

## Setup

```powershell
cd frontend
npm install
copy .env.example .env
```

If your API runs on a different host/port, update `VITE_API_BASE` in `.env`.

## Run

```powershell
npm run dev
```

Open `http://127.0.0.1:5173`.

## Build

```powershell
npm run build
npm run preview
```
